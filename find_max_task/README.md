# Проект: Поиск максимального значения

## Описание
Этот проект содержит реализацию метода для поиска максимального значения в списке.

## Метод findMax

### Синтаксис
```java
public static Integer findMax(List<Integer> numbers)
```

### Параметры
- `numbers`: список целых чисел типа List<Integer>

### Возвращаемое значение
- Максимальное значение в списке (тип Integer)
- `null`, если список пуст или равен null

### Пример использования
```java
List<Integer> numbers = Arrays.asList(1, 5, 3, 9, 2);
Integer max = findMax(numbers);
System.out.println(Максимум:  + max); // Вывод: Максимум: 9
```

### Алгоритм
1. Проверка на null и пустой список
2. Инициализация максимума первым элементом
3. Итерация по остальным элементам
4. Сравнение и обновление максимума при необходимости
5. Возврат результата

## Запуск проекта
```bash
javac src/Main.java
java -cp src Main
```
