# Проект: Поиск максимального значения

## Описание
Программа для поиска максимального значения в списке чисел с вводом от пользователя.

## Запуск
\`\`\`bash
javac src/Main.java
java -cp src Main
\`\`\`

## Пример
\`\`\`
Введите количество элементов в массиве: 3
Введите 3 элементов:
Элемент 1: 10
Элемент 2: 5
Элемент 3: 8
Максимальное значение в массиве: 10
\`\`\`
