Метод findMax:
----------------
Назначение: Находит максимальное значение в переданном списке целых чисел.

Параметры:
- List<Integer> numbers: список целых чисел для поиска
- Integer: максимальное значение или null, если список пуст или равен null
1. не пуст ли список и не равен null
2. назначает переменную max первым элементом списка
3. проходит по всем остальным элементам списка
4. если текущий элемент больше max, обновляет max
5. возвращает найденное максимальное значение
